# 爱酱MOD管理器 · 快速上手

> 5 分钟用起来。完整版见 [`使用说明.md`](./使用说明.md),出问题翻 [`故障排查.md`](./故障排查.md)。

---

## 30 秒流程

```
1. 安装   →   2. 选游戏 + 设路径   →   3. 拖 MOD 进窗口
                                              ↓
                4. 拨开关启用   →   5. 点「启动」开游戏
```

---

## 主界面

| 位置 | 干什么 |
|------|--------|
| 顶部「游戏」下拉 | 切游戏,各游戏 MOD 互不干扰 |
| 顶部「导入」 | 加 MOD(或直接把文件拖进窗口) |
| 顶部「启动」 | 一键开游戏 |
| 左侧分类栏 | 给 MOD 归类(发型/武器/服装等) |
| 中间列表 | 你的全部 MOD,**右侧开关启停** |
| 右上 ⚙️ | 设置 / 部署方式 / 路径 |

---

## 三件常用事

**① 加 MOD** — 直接拖文件进窗口
> 支持 `.zip` / `.rar` / `.7z` / `.pak` / `.utoc` / `.ucas` / `.dll` / `.ini` / `.json` / `.cfg`,套娃压缩包也行。

**② 启用 / 禁用** — 列表右侧拨开关
> 打开 = 装入游戏(原文件自动备份);关闭 = 撤出还原。
> 多选后右键可批量启停或删除;拖动调顺序,**越靠上优先级越高**。

**③ 删除 MOD** — 先关开关再删,程序会先清理部署再删包。

---

## 出问题怎么办

| 现象 | 操作 |
|------|------|
| 启用中程序崩了 | 重开程序,会自动弹「恢复」窗口,按提示选回滚 / 标记失败 |
| MOD 没生效 | 管理中心 → 看「安装记录」/「部署预览」/「冲突分析」 |
| 想报 bug | 管理中心 → 「导出诊断包」(已自动脱敏)→ 发邮箱 |

**重要**:游戏开起来后**别在管理器里改 MOD**,文件被占用会报错。先关游戏再改。

---

## 文件在哪

| 内容 | 位置 |
|------|------|
| 账号 / 分类 / 方案 | `%APPDATA%\UEModManager\` |
| MOD 包仓库 | 软件目录 `Data\Repository\` |
| 原文件备份 | 软件目录 `Data\Backups\` |
| 出错日志 | 软件目录 `console.log` |

**换电脑 / 重装系统**:复制 `%APPDATA%\UEModManager\` 整个目录到新机器同位置即可。

---

## 老版本升级

双击安装目录里的 **`一键迁移老版本数据.bat`**,按弹窗下一步即可。脚本会先完整备份再迁移,稳。
