📥 安装指南

系统要求

项目 最低配置 推荐配置
操作系统 Windows 10 64位 Windows 11 64位
.NET 运行库 .NET 8.0 Runtime .NET 8.0 Runtime（自动安装）
磁盘空间 50 MB 100 MB（含缓存）
内存 2 GB 4 GB

下载软件

  1. 选择下载渠道

    访问官网下载页面，选择以下任意渠道：

    • 百度网盘（提取码：6679）
    • 迅雷网盘
    • UC网盘
    • 夸克网盘
    • GitHub Releases（国际用户推荐）
  2. 下载安装包

    下载 UEModManager_v1.7.37_Setup.exe 安装包（约 30 MB）

  3. 验证文件完整性

    确保下载的文件大小正确，避免下载不完整导致安装失败。

安装步骤

  1. 运行安装程序

    双击 UEModManager_v1.7.37_Setup.exe，如果出现 Windows 安全提示，点击"更多信息" → "仍要运行"。

    ⚠️ 注意：软件未签名可能会触发 Windows Defender 警告，这是正常现象。软件完全开源且安全。
  2. 选择安装路径

    默认安装路径：C:\Program Files\UEModManager

    建议使用默认路径，避免权限问题。

  3. 选择组件
    • ✅ 主程序（必选）
    • ✅ 桌面快捷方式（推荐）
    • ✅ 开始菜单快捷方式（推荐）
  4. 完成安装

    点击"安装"按钮，等待安装完成（约 10-30 秒）。

    安装完成后，选择"立即运行"启动程序。

✅ 安装成功！首次运行会自动安装 .NET 8.0 运行库（如果系统未安装），请耐心等待。

🚀 首次使用

初始化配置

  1. 选择游戏

    首次启动时，软件会自动扫描已安装的虚幻引擎游戏。如果未检测到，可以手动添加：

    • 点击顶部"设置"按钮
    • 点击"添加游戏"
    • 浏览到游戏根目录（包含 .exe 文件的文件夹）
    • 点击"确定"
  2. 设置MOD目录

    软件会自动定位游戏的 ~mods 文件夹，通常位于：

    游戏根目录\游戏名\Content\Paks\~mods
    💡 提示：如果游戏未创建 ~mods 文件夹，软件会自动创建。
  3. 云端登录（可选）

    如需使用云端同步功能，可以注册/登录账号：

    • 点击右上角"登录"按钮
    • 使用邮箱注册或登录
    • 登录后可同步MOD配置到云端
    ⚠️ 注意：不登录也可正常使用所有本地功能，云端登录仅用于跨设备同步。

界面语言

软件默认为中文界面，如需切换语言：

  • 点击"设置" → "语言"
  • 选择"English"或"中文"
  • 重启软件生效

🖥️ 界面介绍

主界面布局

主界面分为四个区域：

1️⃣ 顶部工具栏

  • 游戏选择器：切换不同游戏
  • 刷新按钮：重新扫描MOD文件夹
  • 冲突检测：一键检测MOD冲突
  • 备份管理：管理MOD配置备份
  • 设置：软件设置和游戏配置
  • 登录/账户：云端账户管理

2️⃣ 左侧分类栏

  • 全部：显示所有MOD
  • 已启用：仅显示已启用的MOD
  • 未启用：仅显示未启用的MOD
  • 自定义分类：用户创建的分类（面部、人物、武器、服装等）

右键分类可以：重命名、删除、设置图标

3️⃣ 中间MOD列表

显示当前分类下的所有MOD，每个MOD卡片包含：

  • 预览图：MOD的预览图片（可自定义）
  • MOD名称：文件名或自定义名称
  • 启用开关：快速启用/禁用MOD
  • 标签：MOD所属分类
  • 冲突图标：红色感叹号表示有冲突

4️⃣ 右侧详情面板

选中MOD后显示详细信息：

  • 自定义名称：可编辑的显示名称
  • 描述：MOD描述信息
  • 文件信息：文件大小、修改时间
  • 分类标签：可添加多个分类
  • 操作按钮：删除、打开文件夹、复制路径等

🎯 安装MOD

方法一：自动导入（推荐）

  1. 下载MOD文件

    从MOD网站（如 Nexus Mods、GameBanana）下载 .pak 或 .zip 文件。

  2. 拖放安装

    将 .pak 文件直接拖放到软件主界面。

    软件会自动将文件复制到 ~mods 文件夹并刷新列表。

    ✅ 提示：如果是 .zip 或 .rar 压缩包，软件会自动解压并提取 .pak 文件。
  3. 启用MOD

    在MOD列表中找到刚导入的MOD，点击"启用"开关。

    启用后MOD文件会被重命名为 _P 结尾（部分游戏需要）。

方法二：手动复制

  1. 定位MOD文件夹

    在软件中点击"设置" → "打开MOD文件夹"，或手动导航到：

    游戏根目录\游戏名\Content\Paks\~mods
  2. 复制MOD文件

    将下载的 .pak 文件复制到 ~mods 文件夹。

  3. 刷新列表

    回到软件，点击顶部"刷新"按钮，新MOD会出现在列表中。

支持的文件格式

格式 说明 处理方式
.pak 虚幻引擎MOD包 直接复制
.zip 压缩包 自动解压提取 .pak 文件
.rar 压缩包 自动解压提取 .pak 文件
.7z 压缩包 自动解压提取 .pak 文件
⚠️ 注意：部分MOD可能包含多个 .pak 文件，请根据MOD说明选择正确的文件。

⚙️ 管理MOD

启用/禁用MOD

有两种方式启用或禁用MOD：

  • 快速开关：点击MOD卡片上的开关按钮
  • 右键菜单：右键MOD → 启用/禁用
💡 原理：禁用MOD会在文件名前加 .disabled 后缀，游戏不会加载该文件。启用时移除该后缀。

自定义MOD信息

  1. 修改显示名称

    选中MOD → 右侧详情面板 → "自定义名称"文本框 → 输入新名称 → Enter 保存

    ✅ 提示：自定义名称仅影响软件内显示，不会修改实际文件名。
  2. 添加描述

    在"描述"文本框中输入MOD的详细说明，方便日后查找。

  3. 更换预览图

    点击预览图 → "选择图片"按钮 → 选择本地图片文件 → 确定

    支持格式：.jpg, .png, .gif

  4. 添加标签

    在"分类标签"区域点击"+" → 选择或创建分类 → 确定

    一个MOD可以属于多个分类。

删除MOD

有两种删除方式：

  • 软删除（推荐）：右键MOD → "移至回收站"，可从系统回收站恢复
  • 永久删除：右键MOD → "永久删除"，无法恢复
⚠️ 警告：永久删除无法撤销！建议使用软删除或先备份。

批量操作

按住 Ctrl 或 Shift 键选择多个MOD，然后可以：

  • 批量启用/禁用
  • 批量添加到分类
  • 批量删除
  • 批量导出

🗂️ 分类管理 NEW

创建分类

  1. 新建分类

    在左侧分类栏空白处右键 → "新建分类"

    或点击分类栏底部的 "+" 按钮

  2. 命名分类

    输入分类名称（如"我的最爱"、"高画质"等）

  3. 设置图标（可选）

    右键新分类 → "设置图标" → 选择图标文件

重命名分类

右键分类 → "重命名" → 输入新名称 → Enter 确认

💡 v1.7.37 新特性：重命名分类只修改显示名称，不会影响MOD的标签数据，更加安全！

删除分类

右键分类 → "删除分类"

软件会提示：

  • 如果分类下有MOD，会自动移动到"未分类"
  • 删除后分类会隐藏，不会真的删除
  • 如果MOD重新使用该分类，它会自动显示

拖拽排序

按住分类名称拖动，可以调整分类的显示顺序。

⚠️ 注意："全部"和"未分类"两个默认分类不支持拖拽排序。

移动MOD到分类

三种方式：

  • 拖拽：选中MOD，拖动到左侧分类上
  • 右键菜单：右键MOD → "移动到分类" → 选择目标分类
  • 详情面板：选中MOD → 右侧 → "分类标签" → 添加分类

⚠️ 冲突检测

什么是MOD冲突？

当多个MOD修改了同一个游戏文件或资源时，会产生冲突，导致：

  • 游戏崩溃或无法启动
  • MOD效果不生效或部分生效
  • 画面异常、贴图错误
  • 游戏性能下降

自动冲突检测

  1. 启动检测

    点击顶部工具栏的"冲突检测"按钮

  2. 等待扫描

    软件会分析所有已启用MOD的文件结构，通常需要10-60秒

  3. 查看结果

    检测完成后，冲突的MOD会显示红色感叹号图标

    点击MOD可在详情面板查看冲突详情

解决冲突

根据冲突类型采取不同措施：

1️⃣ 完全冲突（红色）

两个MOD修改了完全相同的文件，只能保留一个：

  • 禁用其中一个MOD
  • 或寻找兼容版本

2️⃣ 部分冲突（橙色）

MOD有部分重叠，可能可以共存：

  • 先尝试同时启用，进游戏测试
  • 如果有问题，调整加载顺序（通过重命名文件）

3️⃣ 潜在冲突（黄色）

可能影响相同游戏系统，建议谨慎：

  • 查看MOD说明，确认是否兼容
  • 逐个启用，进游戏测试效果
💡 专业提示：部分MOD作者会在说明中列出已知冲突列表，优先参考官方说明。

📦 备份还原

为什么要备份？

  • 防止误删重要MOD
  • 保存完美的MOD组合配置
  • 快速切换不同MOD方案
  • 测试新MOD前保存当前状态

创建备份

  1. 打开备份管理

    点击顶部工具栏"备份管理"按钮

  2. 新建备份

    点击"创建备份"按钮

  3. 设置备份信息
    • 备份名称：如"完美配置2025-01-05"
    • 备份描述：记录MOD列表或用途
    • 备份范围：全部MOD / 仅已启用MOD
  4. 开始备份

    点击"确定"，等待备份完成（取决于MOD数量和大小）

还原备份

  1. 选择备份

    在备份列表中选择要还原的备份

  2. 预览备份内容

    点击"查看详情"可以查看备份中包含的MOD列表

  3. 还原备份

    点击"还原"按钮

    ⚠️ 注意：还原会覆盖当前MOD配置，建议先创建当前状态的备份！
  4. 等待还原完成

    软件会自动恢复MOD文件和配置

备份管理技巧

  • 定期备份：每次大改MOD配置前先备份
  • 命名规范：使用日期+用途命名，如"2025-01-05_测试新皮肤"
  • 删除旧备份：定期清理不需要的备份，节省磁盘空间
  • 导出备份：可以将备份文件复制到其他电脑

备份存储位置

%AppData%\UEModManager\Backups\

可以手动复制该文件夹到其他位置进行额外备份。

☁️ 云端同步

功能介绍

云端同步功能允许您：

  • 在多台电脑间同步MOD配置
  • 备份配置到云端，防止本地丢失
  • 分享MOD列表给其他用户
💡 说明：云端同步仅同步配置信息（MOD列表、分类、自定义信息），不上传MOD文件本身。

注册账户

  1. 打开登录窗口

    点击右上角"登录"按钮

  2. 注册账户

    点击"注册新账户"

    • 输入邮箱地址
    • 设置密码（至少8位，包含字母和数字）
    • 输入用户名
  3. 验证邮箱

    检查邮箱收件箱，点击验证链接

    如未收到邮件，检查垃圾邮件文件夹

  4. 登录

    返回软件，输入邮箱和密码登录

同步配置

登录后，软件会自动同步：

  • ✅ 自动上传：每次修改配置后自动上传到云端
  • ✅ 自动下载：启动软件时自动下载最新配置
  • ✅ 冲突解决：本地和云端不一致时，会提示选择保留哪个版本

离线模式

无网络时，软件会自动切换到离线模式：

  • 所有本地功能正常使用
  • 配置修改会在联网后自动同步
  • 不影响MOD的启用/禁用

🎮 剑星 Stellar Blade 专题

游戏信息

游戏名称 Stellar Blade / 剑星
引擎版本 Unreal Engine 4.27
MOD目录 StellarBlade\Content\Paks\~mods
支持版本 标准版 / CNS版（中国特供版）

MOD安装要点

⚠️ 重要：剑星有两个版本（标准版和CNS版），MOD可能不通用，请确认MOD说明中支持的版本。

1️⃣ 文件命名规则

剑星MOD文件必须以 _P 结尾才能被游戏加载：

正确：CharacterMod_P.pak
错误：CharacterMod.pak

软件会自动处理文件命名，启用MOD时自动添加 _P 后缀。

2️⃣ 常见MOD类型

  • 服装MOD：修改角色服装外观
  • 面部MOD：修改角色脸型、化妆
  • 武器MOD：修改武器模型和特效
  • 画质MOD：提升游戏画质、光影效果
  • UI MOD：修改游戏界面

3️⃣ 冲突规则

  • 同一部位的服装MOD会冲突（如两个上衣MOD）
  • 画质MOD之间通常冲突
  • 面部MOD可能与部分服装MOD冲突

推荐MOD站点

  • Nexus Mods - Stellar Blade
  • GameBanana - Stellar Blade

MOD安装要点

1️⃣ UE5 特殊性

  • 无需 _P 后缀
  • 直接将 .pak 文件放入 ~mods 文件夹
  • 游戏会自动加载所有 .pak 文件

2️⃣ 常见MOD类型

  • 皮肤MOD：修改悟空或装备外观
  • 武器MOD：修改金箍棒等武器模型
  • 特效MOD：增强技能特效
  • 难度MOD：调整游戏难度
  • 性能优化：提升帧率、减少卡顿

3️⃣ 注意事项

⚠️ 重要：
  • 部分MOD可能影响游戏平衡性
  • 在线模式下使用MOD可能导致封号风险
  • 建议在单机模式下使用MOD

推荐MOD站点

  • Nexus Mods - Black Myth Wukong
  • 国内MOD论坛（自行搜索）

🔧 常见问题解决

问题1：软件无法启动

症状

双击图标后无反应，或出现错误提示。

解决方案

  1. 确认已安装 .NET 8.0 Runtime（软件会自动提示安装）
  2. 检查Windows版本是否为 Windows 10 64位或更高
  3. 尝试以管理员身份运行
  4. 检查杀毒软件是否拦截（添加到白名单）
  5. 重新安装软件

问题2：检测不到游戏

症状

软件启动后没有显示已安装的游戏。

解决方案

  1. 确认游戏已正确安装
  2. 手动添加游戏路径：设置 → 添加游戏 → 选择游戏根目录
  3. 检查游戏是否为支持的版本（Steam版、Epic版等）

问题3：MOD不生效

症状

启用MOD后，进游戏发现没有效果。

解决方案

  1. 确认MOD已启用（开关为绿色）
  2. 检查MOD文件名是否正确（剑星需要 _P 后缀）
  3. 检查MOD是否与游戏版本匹配
  4. 运行冲突检测，查看是否有冲突
  5. 尝试禁用其他MOD，逐个测试
  6. 查看MOD说明，确认安装方法

问题4：游戏崩溃

症状

安装MOD后游戏无法启动或进入后崩溃。

解决方案

  1. 禁用所有MOD，确认游戏本身可以正常运行
  2. 逐个启用MOD，找出导致崩溃的MOD
  3. 检查MOD文件是否损坏（重新下载）
  4. 使用备份还原功能，恢复到崩溃前的状态
  5. 查看游戏日志文件，确认错误原因

问题5：软件运行缓慢

症状

软件打开慢、切换游戏卡顿。

解决方案

  1. 减少MOD数量（超过100个MOD可能影响性能）
  2. 清理缓存：设置 → 清理缓存
  3. 关闭不必要的后台程序
  4. 将MOD文件夹移动到SSD

Q1: 软件收费吗？

A: 完全免费且开源！所有功能都可免费使用。如果觉得好用，欢迎赞助支持开发。

Q2: 软件是否安全？

A: 完全安全。软件是开源的，代码托管在 GitHub，任何人都可以审查。不会收集任何个人信息（云端同步除外）。

Q3: 支持哪些游戏？

A: 目前支持：

  • 剑星 Stellar Blade（标准版/CNS版）
  • 黑神话：悟空
  • 明末无双
  • 其他虚幻引擎游戏（需手动配置）

我们会持续添加更多游戏支持。

Q4: 使用MOD会被封号吗？

A: 取决于游戏和MOD类型：

  • 纯单机游戏：完全安全
  • 含在线模式游戏：建议只在离线模式使用MOD
  • 竞技游戏：不推荐使用任何MOD

Q5: MOD下载后放在哪里？

A: 两种方式：

  • 直接拖放到软件界面（推荐）
  • 手动复制到 游戏目录\Content\Paks\~mods

Q6: 如何卸载软件？

A: Windows设置 → 应用 → 找到"UE MOD Manager" → 卸载

卸载不会删除游戏MOD文件，需手动清理 ~mods 文件夹。

Q7: 可以分享MOD配置给朋友吗？

A: 可以！两种方式：

  • 导出备份文件，发送给朋友导入
  • 使用云端分享功能（需双方都登录）

Q8: 软件会自动更新吗？

A: 会提示有新版本可用，但不会自动下载安装。您可以选择：

  • 立即下载更新
  • 稍后提醒
  • 忽略此版本
