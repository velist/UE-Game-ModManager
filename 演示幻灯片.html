日常使用

最常用的三件事

01

加 MOD

把文件直接拖进窗口。
支持 zip / rar / 7z / pak / utoc / ucas / dll / ini 等,套娃压缩包也能拆。

02

启用 / 禁用

列表右侧开关。
开 = 装入游戏(自动备份原文件)
关 = 撤出还原(回到干净状态)

03

删除 MOD

先关开关再删。
程序会自动清理部署、撤销备份,绝不留垃圾。

🎯 拖动 MOD 调整顺序,越靠上优先级越高 —— 处理 MOD 冲突就是这么直观。

出问题怎么办

内置智能恢复与诊断工具

🛡️ 启用中崩溃了?
重开程序会自动弹出「恢复」窗口,按提示选择回滚或标记失败 —— 不用手动改文件。
🔍 MOD 没生效?
打开「管理中心」,有完整的安装记录、部署预览、冲突分析,问题一目了然。
📦 想报 bug?
「导出诊断包」自动收集日志(已脱敏隐私),邮件发给我们就行,不用截图描述。
⏪ 改坏了能撤销?
原文件备份在 Data\Backups\,关闭对应 MOD 即可一键还原。
🔁 老版本怎么升级?
双击「一键迁移老版本数据.bat」,脚本先完整备份再迁移,稳。
💻 换电脑怎么办?
把 %APPDATA%\UEModManager\ 整个目录拷到新机器,配置全在里面。
进阶能力

还有这些隐藏功能

冲突智能检测

多个 MOD 修改同一文件时,程序会高亮提示并允许你调整优先级。

方案 / 预设管理

把"一键二次元包"、"硬核难度组"等组合保存成方案,一键切换。

自定义分类

按类型(发型/武器/服装)、来源、作者随心归类,拖拽即可。

实时部署预览

启用前能看清 MOD 会写入哪些文件,避免黑箱操作。

本地密码加密

BCrypt 哈希存储,本地账号也安全;支持邮箱验证码登录。

数据在哪

完全透明的本地存储

所有数据都在你看得见的地方,卸载随时带走。

账号 / 分类 / 方案

%APPDATA%\UEModManager\

SQLite 数据库,跨设备只需复制这一个目录。

MOD 包仓库

软件目录\Data\Repository\

你导入的 MOD 原始文件都在这,可手动整理。

原文件备份

软件目录\Data\Backups\

游戏原版文件,启用 MOD 前自动备份在此。

错误日志

软件目录\console.log

出问题时报 bug 用,内容自动脱敏。
